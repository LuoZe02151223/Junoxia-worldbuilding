# 全球风带（基于泰勒柱模型）

- 本文件描述**不考虑悬浮大陆存在时**全球大气环流的基本框架。

## 一、风带总览（北半球）

> “西风”= 自西向东；“东风”= 自东向西。

| 纬度带     | 风带名称  | 主导风向    | 备注         |
| ------- | ----- | ------- | ---------- |
| 0°–15°  | 赤道喷流  | **西风**  | 稳定的主喷流     |
| 15°–25° | 低纬东风  | **东北风** | 赤道的回流带     |
| 25°–34° | 副热带西风 | **西南风** | 副热带喷流      |
| 34°–42° | 副热带东风 | **东风**  | 副热带喷流的回流带  |
| 42°–49° | 温带西风  | **西风**  | 进入温带后，风带破碎 |
| 49°–55° | 中纬东风  | **东风**  |            |
| 55°–62° | 亚极地西风 | **西风**  | 风暴活动活跃     |
| 62°–68° | 亚极地东风 | **东风**  | 与极区环流衔接    |
| 68°–74° | 极地西风  | **西风**  | 可表现为高纬喷流边界 |
| 74°–80° | 极地东风  | **东风**  | 冷而稳定       |
| 80°–90° | 极冠环流  | 以环极涡旋为主 |            |

上表可以简化为：

```text
1 条宽赤道喷流
+ 2 条副热带风带
+ 4～6 条越来越窄的中高纬喷流
+ 1 个极冠环流
```

## 三、风向

##### 1.赤道喷流（0°–15°）

- **自西向东**，且向赤道辐合上升，是一条宽阔而强劲的西风，风速可达120m/s以上

##### 2. 低纬东风（12°–22°）

- **东北向西南**，以下沉气流为主

##### 3. 副热带西风（22°–32°）

- **西南向东北**

##### 4. 副热带东风（32°–40°）

- **东北向西南**，中纬前缘的一条回流带

##### 5. 40°以上中高纬带

- 多条窄急流交替排列，中间夹杂剪切带与风暴走廊
- **“越往极地，风带越窄、越碎，风向也在相邻纬圈间频繁翻转。”**

## 四、结论

在不受悬浮大陆扰动的理想状态下，朱诺希亚的大气环流以快速自转下形成的纬向急流为骨架。赤道附近是一条极宽的西风主喷流；其外侧的副热带高压背景中并列着数条方向相反的副热带风带。进入温带与高纬后，喷流被进一步分裂为多条交替排列的窄带，越接近极地越显得细密而破碎。
