## 项目简介

**朱诺希亚（Junoxia）** 是一个持续构建中的原创世界观企划。  
它以漂浮于气态巨行星大气层中的大陆与空中生态系统为舞台，围绕：

- 悬浮大陆与大气生态
- 部族、王国与文明形成
- 生物图鉴与生态系统
- 主线正文与别传
- 自然与“第二自然”的关系

展开叙事与设定整理。

本仓库既是作品存档处，也是世界观资料库。

## 建议阅读顺序

如果你是第一次来到这里，可以按下面的顺序阅读：

1. **README 首页**：快速了解企划 
2. **世设**：认识朱诺希亚的基本框架  
3. **正文 / 深谷篇**：已经完成的故事文本
4. **生物图鉴**：基于分类学构建的生物图鉴  
5. **印象板 / assets**：查看视觉参考与氛围素材  
6. **别传与前作**：衍生或早期相关文本
