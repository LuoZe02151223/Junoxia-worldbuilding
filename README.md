## 项目简介

**朱诺希亚（Junoxia）** 是一个持续构建中的原创世界观企划。  
它以漂浮于气态巨行星大气层中的大陆与空中生态系统为舞台，围绕：

- 悬浮大陆与大气生态
- 部族、王国与文明形成
- 生物图鉴与环境分层
- 主线正文与别传
- 神话、自然与“第二自然”的关系

展开叙事与设定整理。

本仓库既是作品档案，也是世界观资料库。
